postcss-grid-fluid

Github

A PostCSS plugin to create fluid grids.

Settings (optional)

Input

@gf {               /* default values */
  width: 1;         /* blob default width */
  gutter: 1.5rem;   /* gutter width */
  display: flex;    /* float or flex */
}

Output

/* Does not output anything. */

Rows

Input

.my-class {
  gf: row 1.5rem;
}

Output

.my-class {
  clear: both;
  margin-right: calc(-1 * 1.5rem);
  display: flex;
  flex-flow: row wrap;
  align-items: flex-start;
  align-content: flex-start;
}

.my-class::after {
  content: "";
  display: table;
  clear: both;
}

Blobs

Input

.my-class {
  gf: blob 1/3 1.5rem flex;
}

Output

.my-class {
  margin-right: 1.5rem;
  flex: 0 1 calc(33.3336% - 1.5rem);
}
