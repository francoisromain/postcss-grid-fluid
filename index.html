postcss-grid-fluid

Github

A PostCSS plugin to create fluid grids.

Settings (optional)

@gf {    /* default values */
  width: 1,          /* width/total of one blob */
  gutter:  1.5rem,   /* width of the gutter */
  display: flex      /* float or flex */  
}

Blobs

.my-class { gf: blob [width](/[total]) ([gutter]) ([display]); }
